# Atividade Prática - Grafos com Neo4j (Tema: Jogos)

Este projeto é uma aplicação Java que utiliza o Neo4j para representar um grafo de relacionamentos entre Jogadores e Jogos. Cumpre todos os requisitos da atividade prática, incluindo o cadastro de vértices, arestas e execução através de Docker.

## Tecnologias Utilizadas
- Java 17
- Maven
- Neo4j (Database de Grafos)
- Docker
